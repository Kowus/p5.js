Example 2-14: Paint with Grays
==============================

This example shows three different gray values on a black back-
ground:

				function setup() {
					createCanvas(480, 120);
				}
				function draw() {
					background(0);
					fill(204);
					ellipse(132, 82, 200, 200);
					fill(153);
					ellipse(228, -16, 200, 200);
					fill(102);
					ellipse(268, 118, 200, 200);
				}
